# WEATHER INSTRUCTION

MODULE PURPOSE:
  Retrieve current weather and 5‑day forecast using OpenWeatherMap.

AVAILABLE TOOL:
  fetch_api

ENDPOINTS:
  - Current weather: http://api.openweathermap.org/data/2.5/weather
  - 5‑day forecast: http://api.openweathermap.org/data/2.5/forecast

REQUIRED PARAMETERS:
  - lat: latitude (float)
  - lon: longitude (float)
  - appid: API token
  - units: "metric"

DEFAULTS:
  - lat = 42.901775
  - lon = -78.972176
  - units = "metric"

TOOL CALL FORMAT:
  <tool_call>
  {"name": "fetch_api",
   "arguments": {
      "url": "http://api.openweathermap.org/data/2.5/weather?lat=42.901775&lon=-78.972176&appid=OPEN_WEATHER_TOKEN&units=metric"
   }
  }
  </tool_call>

AFTER FETCHING:
  - Parse:
      main.temp
      main.feels_like
      main.humidity
      weather[0].description
      wind.speed
      wind.deg
  - Produce a natural conversational summary.
  - Do NOT call any more tools.

ERROR HANDLING:
  - If the tool returns an error, say:
      "Unable to retrieve weather data."
  - Do NOT retry the same failing tool call.
